HTML5 and CSS3 for the Real World - Chapter 8: Transforms and Transitions

  • CSS3 transform property lets you translate, rotate, scale and skew any element on the page. We now have more control over an elements appearance by using transform functions.
  • The transform functions are used by listing them, seperated by a space. They are applied in the order written.
  • transform: translate(x, y) moves an element x from the left and y from the top. translatex and transplatey can be used to move elements vertically or horizontally. Transforms don't work on inline elements so use display: inline-block.
  • Scaling- scale(x, y) scales an element horizontally and then vertically. If only one value is provided then it is used for both, maintaining the origional aspect ratio. scalex and scaley can be used just like with transform.
  • Rotation- positive integers move clockwise, negative integers move counter-clockwise. rotate(#deg)
  • Skew- skew(x, y) can skew the text up and down at an angle. skewx and skewy can also be used.
  • Changing the origin of the transform- by using a transform-origin the figure noticibaly rotates.
  • Choose ordering carefully, the order of transfor functions does matter.

HTML5 and CSS3 for the Real World - Chapter 12: Canvas, SVG, and Drag and Drop

  • To use canvas add a canvas element to the page. Text between canvas tags will only be shown if the browser does not support canvas. Canvas element also takes width and height to determine the coordinate system.
